# 유튜브 소재 발굴기 PRD (Product Requirements Document)

## 1. 개요

### 1.1 제품명
**유튜브 소재 발굴기** (YouTube Content Discovery Tool)

### 1.2 제품 목적
유튜브 크리에이터가 잠재력 있는 콘텐츠 소재를 효율적으로 발굴할 수 있도록 돕는 AI 기반 분석 도구

### 1.3 핵심 가치
- 바이럴 가능성이 높은 영상 패턴 분석
- 댓글 기반 시청자 반응 인사이트 제공
- AI를 활용한 새로운 콘텐츠 아이디어 추천

---

## 2. 주요 기능

### 2.1 영상 검색 기능

#### 2.1.1 키워드 검색
- 사용자가 입력한 키워드로 유튜브 영상 검색
- 한국어 인터페이스 기본 제공

#### 2.1.2 필터 옵션
| 필터 항목 | 설명 |
|----------|------|
| **콘텐츠 유형** | 숏폼(Shorts) / 롱폼(일반 영상) 구분 선택 |
| **바이럴 비율** | 사용자가 원하는 바이럴 비율 범위 설정 가능 |

#### 2.1.3 바이럴 영상 분석
- **바이럴 비율 계산**: `조회수 / 구독자 수` 비율 계산
- 바이럴 비율이 높은 "잘 터진 영상" 자동 식별
- 검색 결과를 바이럴 비율 순으로 정렬

---

### 2.2 댓글 수집 및 분석 기능

#### 2.2.1 댓글 수집
- 선택한 영상의 댓글 자동 수집
- 한국어/영어 혼합 댓글 지원

#### 2.2.2 AI 댓글 분석
댓글 수집 완료 시 자동으로 다음 항목 분석:

| 분석 항목 | 설명 |
|----------|------|
| **시청자 반응 분석** | 댓글에서 나타난 시청자들의 감정/반응 요약 |
| **키워드 빈도 분석** | 자주 언급된 키워드 추출 및 순위 |
| **관심사 파악** | 시청자들이 관심 있어하는 주제 도출 |

---

### 2.3 소재 추천 기능

#### 2.3.1 키워드 추천
- 댓글 분석 결과를 바탕으로 **5개의 추천 키워드** 제공
- 각 키워드별 추천 이유 설명

#### 2.3.2 대본 목차 생성
- 사용자가 5개 키워드 중 **1개 선택**
- 선택된 키워드에 대한 **간단한 대본 목차** 자동 생성

## 4. 기술 요구사항

### 4.1 프론트엔드
- 한국어 기본 인터페이스
- 반응형 웹 디자인
- 직관적인 필터 UI

### 4.2 백엔드
- YouTube Data API 연동
- 댓글 수집 및 처리 로직
- 바이럴 비율 계산 알고리즘

### 4.3 AI/ML
- 댓글 분석용 자연어 처리 (NLP)
- 다국어 (한국어/영어) 텍스트 분석 지원
- 키워드 추출 및 추천 알고리즘
- 대본 목차 생성 AI

---

## 5. UI/UX 요구사항

### 5.1 메인 화면
- 검색창 (키워드 입력)
- 필터 옵션 (숏폼/롱폼, 바이럴 비율)
- 검색 버튼

### 5.2 검색 결과 화면
- 영상 썸네일
- 채널명, 구독자 수
- 조회수, 바이럴 비율
- 댓글 분석 버튼

### 5.3 분석 결과 화면
- 댓글 반응 요약
- 주요 키워드 워드클라우드/리스트
- 추천 소재 키워드 5개
- 키워드 선택 UI

### 5.4 대본 목차 화면
- 선택한 키워드
- 자동 생성된 대본 목차
- 복사/다운로드 기능

---

## 6. 성공 지표 (KPI)

| 지표 | 목표 |
|------|------|
| 검색 정확도 | 바이럴 영상 적중률 80% 이상 |
| 분석 속도 | 댓글 100개 분석 시 10초 이내 |
| 사용자 만족도 | 추천 소재 유용성 평가 4.0/5.0 이상 |
